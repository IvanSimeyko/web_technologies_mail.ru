Сооздание объектов происходит, как и создание экземпляров класса. Нжно не забывать метод save.

QUERYSETS
QuerySet - невыполненный запрос. QuerySet являются ленивыми (lazy) объектами. Это значит, что запрос осуществляется не в момент создания QuerySet, а в момент итерации по нему, либо вызова метода, возвращающего результат.
Есть два типа методов: которые поддержуют chaining (цепочки вызовов) и те, которые возвращают значение.
Chaining - методы возвращадт объект того же самого класса, только измененный.
posts = Post.objects   # object ModelManager
Methods - filter, exclude, order_by (сортировка), reverse, annotate, values (выбор отдельных колонок, а не объектов), distinct (выборка уникальных значений)

Методы результ: create, update, delete, get_or_create, count
