Основные типы запросов:
-запросы статических документов
-запросы динамических документов
-отправлка данных форм
-AJAX-запросы
-запросы к API сайта
-персистентные соединения 

Основные задачи:
- маршрутизация урл
- парсинг заголовков и параметров запроса
- хранения состояния (сессии пользователя)
- выпобнение бизнес ллогики
- работа с базами данных
- генерация HTML страницы и JSON ответа

MVC
Model-View- Controller - структура веб приложения
Middlware - обертки над нашим приложением, которые добавляют в него некоторые функции.

Роли компонентов MVC
Router - ввыббор конкретного controller по url
Model - реализация бизнес-логики приложения
Controller - работа с HTML, связь controler и view
View - генерация HTML или другого представления

DJANGO
Соглашение об именовании: model - model, Router - urls.py, Controller - views, View - templates

Патерн local.settings.py
В настройках веб приложения существуют опции, которые будут меняться редко и опции, которые могут меняться часто в зависимотсти от пользователя, который будет с ними работать. Последнии опции лучше вносить в local_settings.py, который в свою очередь не отслеживать в системе контроля версий.
