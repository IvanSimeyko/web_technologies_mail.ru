HTML
Строка DOXCTYPE - уточняет тип содержимого документа и указыавет HTML парсеру, как правильно разбирать данный документ.

Семантика - в html есть оприсание на то как браузер должен показывать определенный тег. 
Тег верхнего уровня html, head - метаинформация документа, body - основное тело страницы
title - заголовок окна браузера
meta - дополнительная информация для user-agent'ов
link - указание связаннных ресурсов
script - загрузка js (async - указание на то что js загружать в фоне, не останавливая папвинг html)

Блочные и Строчные теги
Блочные - h1...h6, p, hr, pre, blockquote, div. Особенность блочных тегов в том что они занимают всю ширину браузера, всегда начинаются с новой строки. Для блочных тегов можно задавать явно ширину и высоту. 

Строчные тег: a, em (i), strong (b), img, sub - нижний текст, sup - верхний текст, span - абстрактный строчный контейнер (аналог блочного тега div). Особенности - занимают минимальную возможную ширину и не прерывают строчку. Размер строчного тега считается автоматически по его содержимому, соответственно задать его размер невозможно (исключение тег img). 

Формы используют для отправлки данных на сервер. 
Поля ввода формы:
input - универсальное поле, может быть.
    type='hidden' - невидимое
    type='text' - текстовое поле
    type='checkbox' - переключатель да/нет
button - кнопка
textarea - многострочное поле ввода
select, option - выпадающий список
