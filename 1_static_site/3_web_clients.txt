Веб клиент - загружают и отображают документы. Самый популярный браузер.
Во многих языках программирования есть своя библиотека для работы с протоколом HTTP. Эта библиотека тоже веб-клиент.

Консольные веб-клиенты - (например wget) используют для автоматизаиции в некоторых скриптах, для создания "статической копии сайта", для отладки приложений.
Утилита telnet - простейшее соедство отладки. Открывает tcp соединение и связывает его с консолью, позволяя общаться с веб-сервером напрямую с клавиатуры. 

Уоитита curl - позволяет посмотреть заголовки протокола HTTP
curl -vv 'https://stepic.org/favicon.ico'

Браузер.
Существуют ОС и 3D игры работающие внутри браузеров.  
Сценарий работы современного приложения:
-браузер загружает Веб страницу, русурсы и отображает ее
-JavaScript загружает данные с помощью AJAX-запросов
-JavaScript обеспечивает поолный UI на странице
-Пользователь взаимодействует с UI, что приводит к вызову JavaScript обработчиков
-JavaScript обновляет данные на сервере или загружает новые данные используя AJAX
